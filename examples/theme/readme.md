一些用于 elecV2P 的测试主题。

使用方式

1. 下载该目录下的主题文件，比如 elecV2P_theme.json
2. 打开 elecV2P webUI，SETTING/设置->webUI 相关设置->导入常用主题，选择下载的主题文件
3. 点击**预览**进行查看，确定后点击**保存**正式启用

*主题功能仅部分用户可用*
