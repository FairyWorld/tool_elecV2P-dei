### 缓存结构

``` JS
const script_cache = new Map();
script_cache.set('script_name', {
  name: "name",
  time: "2022-04-17 21:25"
  type: "JS",
  grant: true,
  sudo: false,
  compatible: 'nodejs',
  content: CONTEXT.final,
  content: "...",
  done: "res",
})
```

### 目的、优点

- 加快脚本载入速度
- 加快脚本处理速度

### 问题

- 占用内存
- 部分资源无法释放

### 考虑添加

- 极速模式（不运行脚本，直接返回上次执行结果
